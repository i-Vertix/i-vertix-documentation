# Groups

This tab allows you to manage groups for a user (if your own permissions
allow it).

![Global view groups](../../../../assets/modules/administration/users/images/groups_global_view.png)

## Add a group

To add a group :

- select the group in the dropdown list
- if the user can manage the group
- click on Add

![Add group](../../../../assets/modules/administration/users/images/add_group.png)

:::info

If the user is a manager of a group, he has the right to modify the
group. He can therefore add and delete users from this group.

:::

## Delete a group

To delete a group, use massive actions
