# Plugs

This tab shows the number of plugs available on a PDU. This item can be
linked to a [rack](../../../modules/assets/racks.md) or an
[enclosure](../../../modules/assets/enclosures.md)

![View plugs on PDUs](../../../assets/modules/assets/images/pdu-plugs.png)

## Add a plug

- To add a plug on PDU, choose the type of plug you require from the
  drop-down list
- Select a number of plugs available on the PDUs
- Click on **Add**

![Add plugs on PDUs](../../../assets/modules/assets/images/pdu_add_plug.png)

:::info

you can use the same plug model several times

:::

## Delete a plug

To delete a plug, use the massive action :

- Select the plug
- Click on **Actions**
- Select **Delete permanently the relation with selected elements**
- Click on **Post**

![Delete plugs on PDUs](../../../assets/modules/assets/images/pdu_delete_plug.png)

## Create a plug

If the plug model you want is not available, you can create a new one.

- Click on **+**

![Create a new plugs on PDUs](../../../assets/modules/assets/images/pdu_create_plug.png)

- Fill in the fields
- Click on **+Add**

![fill in the fields plugs on PDUs](../../../assets/modules/assets/images/pdu_fill_in_plug.png)

:::info

You can find a list of all your plugs in **Setup \> Dropdowns \>
Plugs**

:::
