# Network

**Network** is information added manually. you can pre-fill this
information using a template.

![network](../../assets/tabs/common_fields/images/network.png)

- To create a network, click **+**
- Enter a **Name**
- Enter a **Comment** (optional)
- Click **+ Add**
- You can add another one or close this window
- Then select the network you want from the drop-down list

![add network](../../assets/tabs/common_fields/images/add-network.png)
