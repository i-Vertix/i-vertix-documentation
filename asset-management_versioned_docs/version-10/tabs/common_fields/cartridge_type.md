# Cartridge Type

This field defines the type of cartridge (laser, inkjet, etc.)

## Add a type of cartridge

- You can add a new item by clicking on **+**
- Add a name
- Add a comment if necessary
- Click on **+ Add**

![add a new cartridge type](../../assets/tabs/common_fields/images/cartridge_type_add.png)

## Delete a type of cartridge

- To delete a cartridge type, list the cartridge type by clicking on
  **i** (show cartridges type) or go to **Setup \> Dropdowns \>
  Cartridge types**

![list a cartridge type](../../assets/tabs/common_fields/images/cartridge_type_list.png)

- Click on the cartridge you want to delete
- Then click on **delete permanently**

:::info

Use massive actions to remove several types of cartridge at the same
time

:::
