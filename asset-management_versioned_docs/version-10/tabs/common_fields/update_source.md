# Update source

Update source is the way in which the machine has been incremented in
the inventory. If it was done by the agent, i-Vertix ITAM Native Inventory will
be indicated. For a manual entry, this field will remain empty. You can
add manual sources

![update source](../../assets/tabs/common_fields/images/update-source.png)

- To create update source, click **+**
- Enter a **Name**
- Enter a comment (optional)
- Close the window and select the value in the drop-down list

![update source](../../assets/tabs/common_fields/images/add-update-source.png)
