# Model

When the inventory is uploaded, the template field is filled in
automatically. You can also create/add templates manually. Models can
also be used by templates and rules.

![computer model](../../assets/tabs/common_fields/images/model.png)

- To create a model, click **+**
- Add a **Name**
- Add a **Product number** if it's necessary
- You can specify the **Weight** and the **Depth**, the **Required
  units** (in the case of a rack server)
- You can indicate the **Power connections** and the **Power
  consumption**
- Indicate whether it is a half rack
- Add pictures (front, rear, other)
