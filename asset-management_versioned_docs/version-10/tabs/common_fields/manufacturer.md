# Manufacturer

Manufacturers are automatically retrieved by the automatic inventory and
assigned to the computer. you can also create/add them manually.

![computer-manufacturer](../../assets/tabs/common_fields/images/manufacturer.png)

## Add a manufacturer

- To create a manufacturer, click **+**
- Add a **Name**
- You can add a registered ID issued by
  [PCI-SIG](https://pcisig.com/developers/integrators-list) for USB
  or/and PCI. You can add more by clicking **+**.

Once the manufacturer has been created, you can add another or select it
from the drop-down list.

![add computer manufacturer](../../assets/tabs/common_fields/images/add-manufacturer.png)

## Delete a manufacturer

To delete a manufacturer, use the massive action with **Actions** button

![delete manufacturer](../../assets/tabs/common_fields/images/delete-manufacturer.png)

Or enter in the group you want and click on **Delete permanently**

![delete manufacturer by delete permaently](../../assets/tabs/common_fields/images/delete-manufacturer-individualy.png)
