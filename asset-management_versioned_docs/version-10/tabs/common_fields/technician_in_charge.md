# Technician in charge

A technician in charge can modify the information of the computer and
his elements (status, computer type, etc.)

For users created from an AD/LDAP(S) or SCIM, please refer to your
internal procedures.

:::info

A technician is a person with a technician profile or higher

:::

- In **Technician in charge** tab, select the technician you require.

![technician in charge](../../assets/tabs/common_fields/images/asset-technician-in-charge-1.png)

A technician in charge can see the material that he can manage in
**Administration \> Users** in the **managed items** in the user file

![technician in charge user file](../../assets/tabs/common_fields/images/asset-technician-in-charge.png)

To create or import a user, go to
[users](../../modules/administration/users)
